# KimuVerb User Manual

KimuVerb brings ocean acoustic character to your music production. It combines advanced reverb processing with an immersive visual interface.

## Getting Started
1. Load KimuVerb on any audio track
2. Select a factory preset or start from scratch
3. Adjust Mix to blend wet/dry signals
4. Fine-tune Size and Decay for space characteristics
5. Use Depth for modulation and movement

## Advanced Techniques

### Sidechain Compression
Place KimuVerb on a send track and sidechain the reverb return to the dry signal for ducking effects.

### Automation
Automate Size and Depth for evolving spaces that change across an arrangement.

### Layering
Combine multiple instances with different settings for complex, multi-layered reverb effects.
