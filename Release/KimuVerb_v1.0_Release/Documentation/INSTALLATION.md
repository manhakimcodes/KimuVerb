# KimuVerb Installation Guide

## Installation Steps

### Windows Installation
1. Download the KimuVerb installer from a trusted source
2. Run the installer as Administrator
3. Follow the installation wizard prompts
4. Choose your VST3 plugin directory (default recommended)
5. Complete the installation

## Troubleshooting

### Plugin Not Found
- Ensure the VST3 folder is correctly set in DAW preferences
- Run a DAW plugin rescan
- Check Windows security permissions

### GUI Issues
- Update graphics drivers
- Disable GPU acceleration in your DAW
- Verify OpenGL 3.3+ support

## System Requirements

### Minimum Requirements
- Windows 10 (64-bit)
- 4GB RAM
- OpenGL 3.3 compatible graphics
- VST3-compatible DAW

### Recommended Requirements
- Windows 11 (64-bit)
- 8GB+ RAM
- Dedicated graphics card
- SSD for faster loading
