# KimuVerb v1.0.4 Release Notes

## Fixes
- Pre-delay phase fix (shared smoothing across channels)
- Removed dry-path filtering to prevent comb filtering
- Improved wet/dry mix formula to reduce phase cancellation
- Removed limiter and added gentle soft clip to prevent hard clipping
- Reduced UI lag while dragging/moving the plugin window

## Included
- Updated installer and release zip
- Updated docs and GitHub Pages site

## System Requirements
- Windows 10/11 (64-bit)
- VST3-compatible DAW
- OpenGL 3.3+ capable graphics
- 4GB RAM minimum (8GB recommended)
