# KimuVerb v1.0.1 Release Notes

## Fixes
- Installer now places KimuVerb.vst3 directly in C:\Program Files\Common Files\VST3 for reliable DAW detection
- Reduced UI lag while dragging/moving the plugin window
- Fixed double filtering on the wet path to reduce phasing artifacts

## Included
- Updated installer and release zip
- Updated docs and GitHub Pages site

## System Requirements
- Windows 10/11 (64-bit)
- VST3-compatible DAW
- OpenGL 3.3+ capable graphics
- 4GB RAM minimum (8GB recommended)
